How a website works

A DNS Server is like a phonebook with all the domains in the world and the IP addresses of the servers that holds the required information of that domain. A browser uses the IP address to find the server holding the required files(html, css, javascript) to display the website on the user’s window.

IP address IP address is basically a unique individual name assigned to every server computer when it is added to the Internet/ cloud. Just like every single house has a specific residence address and if you want to send a letter, you mention the residential address in your letter and the mail man will know where to deliver that mail to. In a prallel way, the browser can find the website’s server from the internet/ cloud by scanning for the website’s IP address.

Internet/ Cloud is a bunch of computers connected together and each computer has a unique IP address which holds the required files and database for it’s domain.
